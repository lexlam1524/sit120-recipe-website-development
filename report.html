Overall description:

This website is the recipe application website. You can find recipe in this website and search what you want.

Design: the website consists of five parts which are home, search page, login page, signup page and recipe page. It is beginning in home page; you can assign to different page with the button on top bar or click the photo of the food.


On the home page, there are different recipe, and search engine. You can search any recipe in the search bar, it will link to search page.


On the search page, you also can search any recipe you want


On the login page, you can fill in your account detail to login, it also can link to the sign-up page.


On the signup page, you can fill in your account detail to signup, it also can link to the login page.


On the recipe page, you can see the description, step, ingredient, and tool that you need. Also there are a count function that you can give a link to the recipe.
